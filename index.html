SP-110: Green Linux User Anomaly Detection System

Project Overview

The Linux User Anomaly Detection System is designed to monitor and identify unusual log patterns in Linux environments. This system aims to enhance security by detecting potential threats through machine learning techniques like Isolation Forest.
